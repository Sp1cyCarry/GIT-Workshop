# ***Here are some basic markup symbols to help you understand and style your ReadMe!***

# __Markdown Style Cheatsheet__

```markdown
# HEADINGS
# H1
// biggest heading

## H2
// second level heading

### H3
// third level heading


# TEXT STYLES
**bold**
// bold text

*italic*
// italic text

***bold italic***
// bold + italic


# LISTS
- item
// unordered list

1. item
// ordered list


# CODE
`inline code`
// small code snippet

```
**To display the code you use "```"**, you need to open and close this tag to visually show your code snippet.
// code block (no language)
```

# LINKS
[link text](https://example.com)
// clickable link


# IMAGES
![alt text](image.png)
// displays an image


# BLOCKQUOTES
> quote
// blockquote

>> nested quote
// quote inside a quote


# HORIZONTAL RULES
---
// horizontal line

***
// alternative horizontal line

___
// another variant


# TABLES
| Column 1 | Column 2 |
|----------|----------|
| value    | value    |
// basic table

# CHECKBOXES
- [ ] todo
// unchecked

- [x] done
// checked
```
